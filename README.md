Manipulating hierarchical BST tree structures

## Installation

```sh
npm install i-like-bonsai
```

## Usage

```javascript
import bonsai from 'i-like-bonsai'

// const nodeId = Tree.newNodeID()
const tree = bonsai(rootNodeId)
```
